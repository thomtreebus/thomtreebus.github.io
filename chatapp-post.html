ChatApp - Messaging

My goal for creating this project was to succesfully connect Firebase with chatengine.io and integrate both Facebook and Google Authentication. The final product is a responsive React Application.

New users can sign in to an account using either Google or Facebook authentication. Once signed in, a profile name and avatar will be created for them using their Google/Facebook credentials.

Users can create chats and invite an individual or a group of people to the chat.

Both messages and images can be sent as chats. Users can also see when other people in the chat have seen their messages.

Users can sign out using the button at the top right of the application which will bring them back to the sign-in screen.

Technologies:
  • - React JS
  • - Firebase
  • - Chat Engine
  • - Javascript
  • - React Context API
  • - REST APIs
