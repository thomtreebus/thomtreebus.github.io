Memories Project

View the live version

This was a project I created once I completed a web development course on Udemy.

The app lets you create a memory/post with a title, description, image, and tags.

Once a post is created, it will be displayed in a nice looking card. The user can edit any of the post's attributes and save their edits.

Posts can also be liked or deleted.

I worked on this system for 2 years before selling this to FOI Laboratories.

Technologies:
  • - MongoDB
  • - Express REST Framework
  • - React JS
  • - Node JS
  • - Redux
